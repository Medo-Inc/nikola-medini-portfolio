IHF U19 World Championship Croatia 2023 Graphics Package

About the project

This project was made for IHF U19 World Championship Croatia 2023 . I had to design and animate UI elements for the U19 tournament. The package consisted out of Line up, Half-time, Intro, Game Statistics, Referees and Score graphics. I was only given a logo and a font, and I had to create everything else by myself. I used the logo as my target color palette.

Logo

Color Palette

I wanted my design to have a consistency with the given logo. I created a few design styles and proposed them to the client. While Client was his favorite style, I created a few animations so he could better understand how style frames would move in real time. After the client picked his favorite, I began creating other designs found in the package.

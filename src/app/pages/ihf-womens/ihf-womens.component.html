UI/UX + Motion Design + Concept Idea

IHF Women's World Championship 2023 Graphics Package

About the project

This project was made for IHF Women's 2023 World Championship Croatia 2023 . I had to design and animate UI elements. The package consisted out of Line up, Half-time, Intro, Game Statistics, Referees and Score graphics. I was only given a logo and a font, and I had to create everything else by myself. I used the logo as my target color palette.
